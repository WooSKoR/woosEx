jQuery - travering(= DOM tree)

특정요소 찾기 , 선택한 요소를 기준으로 위, 아래, 옆으로 이동하면서 선택

상위객체 - jQuery Ancestors

  • parent() : 선택한요소의 부모 객체
  • parents() : 선택한요소의 모든 부모 객체
  • parentsuntil() : 선택한 요소와 ()요소 사이의 상위객체 전부

하위객체 - jQuery Decendants

find() : 선택한 요소의 하위 객체 전부
children() : 선택한 요소의 직접적인 하위객체 (자손의 자손은 선택이안됨)

형제객체 : jQuery Sibilings

  1. siblings() : 선택한 요소의 형제객체(자기자신 제외)
  2. next() : 선택한 요소의 다음객체
  3. nextAll() : 선택한 요소의 다음 전부
  4. nextUntil() : 선택한 요소의 ()사이의 동생객체
  5. prev() : 선택한 요소의 전 객체
  6. prebAll() : 선택한 요소의 전 전부
  7. prevUntil() : 선택한 요소의 ()사이의 형 객체
