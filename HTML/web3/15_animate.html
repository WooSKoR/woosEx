animate()

스크립트로 움직이는 동작을 적용

$('선택자').animate({ 스타일 }, duration, easing, 콜백함수);

*스타일 = 속성 : '값'

  • 크기조절 : width / height
  • 위치조절 : top / right / bottom / left
  • 여백조절 : margin / padding
  • 테두리 : border / borderRaduis
  • 글자 : fontSize / lineHeight...
  • <투명도 : opacity/li>

animate적용

duration 재생속도

움직임 동작을 완료할때까지 걸리는 시간(=재생시간)

milliseconds 사용 (1초 = 1000)

400이 기본값 (0.4초) / 'fast' = 200 / 'slow' = 600

easing

= timing-function 수치변형함수

'swing' 천천히 시작해서 빨라졌다가 느려짐(기본값)

'linear' 첨부터 끝까지 일정한속도

콜백함수

animate() 동작이 끝난 후 실행되어질 값
